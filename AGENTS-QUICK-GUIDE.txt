================================================================================
  WALIYA v1.0.0 — Agent Quick Guide (REAL SHOPS)
================================================================================

INSTALL (once per hall PC)
--------------------------
1. Run Waliya-Setup-1.0.0.exe
2. Install -> open Waliya from Desktop

FIRST TIME ON THIS PC (REQUIRED)
--------------------------------
  Your admin creates your account on THEIR admin PC, then sends you:
    - Username (e.g. abebe)
    - Password
    - TAS- setup code (long code starting with TAS-)

  On YOUR hall PC:
    1. Open Waliya
    2. Login screen -> click "Activate PC"
    3. Paste the full TAS- code from admin
    4. Click "Activate this PC"
    5. Then click "Sign in" and use your username + password

  Without the TAS code, login will NOT work on a new PC.

RECHARGE WALLET (after you pay admin)
-------------------------------------
  1. Pay admin cash or Telebirr
  2. Admin sends you a TBG-... code by SMS (for YOUR username only)
  3. Recharge menu -> paste code -> Recharge
  4. Code works once

  If error "wrong organization key":
    You probably skipped Activate PC. Go back to login -> Activate PC
    with your TAS code, then ask admin for a NEW TBG code.

RUN A GAME
----------
  1. Game Board
  2. Set bet, Language Amharic, select cartella numbers (blue)
  3. Start Game
  4. Press green START button -> numbers 1-75 call with voice
  5. STOP to pause, RESUME to continue
  6. BINGO! -> verify cartella -> End Game

  Cartella voice: when you tap a cartella number, you hear "ካርቴላ …"
  (mute/unmute on the game board). Uses bundled Amharic clips offline.

VOICE QUALITY (optional — for your own recordings)
--------------------------------------------------
  See public/audio/README.txt for folder layout and alternative voices.
  npm run recording:script   -> checklist of all phrases
  Put each voice pack in:    public/audio/voices/male1/  (or male2, female1, female2)
  npm run validate:audio && npm run pack:win

------------
  Windows 8, 8.1, 10, or 11 (64-bit)
  No internet needed after install

HELP
----
  Contact your admin for TAS setup code, login, recharge codes, password reset.
