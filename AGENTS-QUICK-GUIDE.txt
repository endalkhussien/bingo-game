================================================================================
  TEBIB-BINGO v1.0.0 — Agent Quick Guide (REAL SHOPS)
================================================================================

INSTALL (once per PC)
---------------------
1. Run TEBIB-Bingo-Setup-1.0.0.exe
2. Install -> open TEBIB-Bingo from Desktop

YOUR LOGIN (from your admin — NOT demo)
---------------------------------------
  Your admin gives you:
    Username: (e.g. abebe)
    Password: (your password)

  First time on this PC only:
    Admin may login as admin/admin123 and create your account here
    with the SAME username and password.

  Demo (testing only): agent / agent123

RECHARGE WALLET (after you pay admin)
-------------------------------------
  1. Pay admin cash or Telebirr
  2. Admin sends you a TBG-... code by SMS
  3. Recharge menu -> paste code -> Recharge
  4. Code works once, for your username only

  If error "wrong organization key":
    Settings -> paste organization key from admin -> Save

RUN A GAME
----------
  1. Game Board
  2. Set bet, Language Amharic, select cartella numbers (blue)
  3. Start Game
  4. Press green START button -> numbers 1-75 call with voice
  5. STOP to pause, RESUME to continue
  6. BINGO! -> verify cartella -> End Game

REQUIREMENTS
------------
  Windows 8, 8.1, 10, or 11 (64-bit)
  No internet needed after install

HELP
----
  Contact your admin for login, recharge codes, and password reset.
